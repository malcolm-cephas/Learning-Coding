# Extra Programming Learning Lab

A multi-disciplinary repository containing hundreds of programs, experiments, and projects for advanced programming education.

### 📜 Master Lab Directory

Explore the various disciplines in this lab through their respective clickable directories:

*   **[Python/](Python/)** - Core Python, Artificial Intelligence, Machine Learning, and NLP.
*   **[C++/](c++/)** - Object-Oriented Programming, Templates, and Exception Handling.
*   **[C/](c/)** - Mathematical computing, Pointer-based Stats, and Structs.
*   **[Java/](Java/)** - GUI, Multi-threading, Data Structures, and JDBC.
*   **[Node JS/](Node%20JS/)** - Full-stack development, Express, React, and REST APIs.
*   **[Scripting Languages/](Scripting%20Languages/)** - Advanced Ruby, TCL, and Perl programming.
*   **[Prolog/](Prolog/)** - Logic programming, Expert systems, and AI puzzles.
*   **[Web Programming/](web%20programming/)** - PHP, AJAX, XML, and DHTML web technologies.
*   **[Computer Networks/](computer%20networks/)** - C-based networking protocols and traffic simulations.
*   **[R Programming/](r%20programming/)** - Statistical modeling and data visualization in R.
*   **[Data Mining/](data%20mining/)** - WEKA toolkit experiments and ARFF datasets.
