# Java Programming Lab

A collection of Java programs covering GUI (Swing/AWT), Multi-threading, Exception Handling, Data Structures, and File I/O.

### 📜 Master Program List

1.  **[SimpleDemo.java](SimpleDemo.java)**: Basic Java program showcasing `if-else` and `for` loop logic.
2.  **[Calculator.java](Calculator.java)**: Simple calculator using `GridLayout` and `Swing`.
3.  **[AppletDemo.java](AppletDemo.java)**: Java Applets for displaying messages and computing factorials.
4.  **[DivisionGUI.java](DivisionGUI.java)**: UI for integer divisions with robust exception handling.
5.  **[MultiThreadDemo.java](MultiThreadDemo.java)**: Multi-threaded application for random number generation and processing.
6.  **[DoublyLinkedList.java](DoublyLinkedList.java)**: Implementation of a doubly linked list with deletion and display features.
7.  **[TrafficLight.java](TrafficLight.java)**: Simulated traffic light with radio button controls and dynamic status messages.
8.  **[ShapesArea.java](ShapesArea.java)**: Demonstrating abstraction and inheritance with `Shape`, `Rectangle`, `Triangle`, and `Circle`.
9.  **[TableDisplay.java](TableDisplay.java)**: Read a table from a CSV file (`Table.txt`) and display it using `GridLayout`.
10. **[MouseEventsDemo.java](MouseEventsDemo.java)**: Handle all mouse events and display event names using `Adapter classes`.
11. **[PhonebookHash.java](PhonebookHash.java)**: Load names and numbers from a file and search using Hash Tables.
12. **[ProducerConsumer.java](ProducerConsumer.java)**: Implementation of the classic Producer-Consumer problem.
13. **[FileListing.java](FileListing.java)**: Recursive directory and file listing utility.
14. **[OOPDemo.java](OOPDemo.java)**: Demonstrating Encapsulation, Inheritance, Polymorphism, and Abstraction.
15. **[ExceptionDemo.java](ExceptionDemo.java)**: Comprehensive handling of Checked, Unchecked, and Custom Exceptions.
16. **[RandomAccessFileDemo.java](RandomAccessFileDemo.java)**: Direct access reading and writing operations on files.
17. **[CollectionsDemo/](CollectionsDemo/)**: A package exploring List, Set, and Map collection frameworks.
18. **[ThreadSyncDemo.java](ThreadSyncDemo.java)**: Synchronized ticket booking simulation for concurrent systems.
19. **[StudentJDBC.java](StudentJDBC.java)**: Database interaction for Student CRUD operations via JDBC.
20. **[Biggest3.java](Biggest3.java)**: Finding the largest of three numbers using logical operators.
21. **[Fibo.java](Fibo.java)**: Printing the first 10 numbers in the Fibonacci sequence.
22. **[Factorial.java](Factorial.java)**: Iterative calculation of the factorial of a given number.
23. **[Stars.java](Stars.java)**: Printing a right-angled star pattern using nested loops.
24. **[SumDigits.java](SumDigits.java)**: Calculating the sum of all digits in a given integer.
25. **[Swap.java](Swap.java)**: Swapping two numbers using a temporary variable.
26. **[Prime.java](Prime.java)**: Generating prime numbers up to a specified limit.
27. **[Palin.java](Palin.java)**: Checking if a string is a palindrome using `StringBuffer`.
28. **[SeriesFact.java](SeriesFact.java)**: Calculating the sum of the series $1/1! + 1/2! + 1/3! ...$
29. **[Quadratic.java](Quadratic.java)**: Solving quadratic equations using command line arguments.
