# Artificial Intelligence (Prolog Lab)

A collection of AI algorithms, search strategies, and classic puzzles implemented in Prolog.

### 📜 Master Program List

1.  **[FactsQueries.pl](FactsQueries.pl)**: Implementation of simple facts and queries.
2.  **[Arithmetic.pl](Arithmetic.pl)**: Demonstration of simple arithmetic operations in Prolog.
3.  **[MonkeyBanana.pl](MonkeyBanana.pl)**: Solving the classic Monkey-Banana AI problem.
4.  **[Hanoi.pl](Hanoi.pl)**: Recursive solution for the Tower of Hanoi.
5.  **[Puzzle8.pl](Puzzle8.pl)**: Framework for solving 8-puzzle problems.
6.  **[Queens4.pl](Queens4.pl)**: Solving the 4-Queens combinatorial problem.
7.  **[TSP.pl](TSP.pl)**: Representation of the Traveling Salesman problem.
8.  **[WaterJug.pl](WaterJug.pl)**: Heuristic state-space search for the Water Jug puzzle.
