# Prolog Programming Lab

A collection of Prolog programs based on search algorithms, logic programming, and AI problem solving.

### 📜 Master Program List

1.  **[LogicFacts.pl](LogicFacts.pl)**: Simple logic facts (Likes, Gender, Ownership).
2.  **[Temperature.pl](Temperature.pl)**: Temperature conversion and freezing checks.
3.  **[MonkeyBanana.pl](MonkeyBanana.pl)**: AI puzzle solving: The Monkey and Banana problem.
4.  **[MedicalDiagnosis.pl](MedicalDiagnosis.pl)**: Expert system for diagnosis with Red/Green cuts.
5.  **[Queens4.pl](Queens4.pl)**: Solving the 4-Queens combinatorial problem.
6.  **[TravelingSalesman.pl](TravelingSalesman.pl)**: Path optimization and graph search.
7.  **[WaterJug.pl](WaterJug.pl)**: Solving the state-space water jug puzzle.
8.  **[ListOperations.pl](ListOperations.pl)**: Nth item removal and insertion in lists.
9.  **[BinaryTree.pl](BinaryTree.pl)**: Binary search tree (BST) operations with leaf addition.
10. **[AccumulatorCounts.pl](AccumulatorCounts.pl)**: Using accumulators to count lists.
11. **[MemCount.pl](MemCount.pl)**: Counting occurrences without accumulators.
