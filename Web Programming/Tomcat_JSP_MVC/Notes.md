# Tomcat, Servlets, and JSP Integration

### 📋 Installation (Exp 12/8)
1.  Download **Apache Tomcat** (e.g., version 10).
2.  Set `CATALINA_HOME` environment variable.
3.  Deploy WAR files to the `webapps/` folder.

### 🍪 Servlets & Cookies (Exp 8)
- Use `HttpServlet` class for dynamic content.
- Use `Cookie` class to store user IDs or Shopping Cart IDs.
- Store configuration in `WEB-INF/web.xml`.

### 🏗️ MVC and JSP (Exp 9)
- **Model**: Database connection and DAOs.
- **View**: JSP pages for display logic.
- **Controller**: Servlets that handle requests and call the Model.
- **Database**: Load book catalogue dynamically (MySQL).
