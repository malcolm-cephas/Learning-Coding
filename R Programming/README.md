# R Programming Lab

A collection of R scripts covering basics, data visualization, and statistical modeling.

### 📜 Master Program List

1.  **[R_Basics.R](R_Basics.R)**: Installation, variables, and setup.
2.  **[VectorMatrixOps.R](VectorMatrixOps.R)**: Mathematical operations on vectors.
3.  **[DataSubsetting.R](DataSubsetting.R)**: Indexing techniques and 3x3 matrix creation.
4.  **[PieCharts.R](PieCharts.R)**: Visualization using Pie charts (2D and 3D).
5.  **[StatisticalPlots.R](StatisticalPlots.R)**: Box, Hist, Line, and Scatter plots.
6.  **[DataStructures.R](DataStructures.R)**: Implementation of Vectors, Lists, and Data Frames.
7.  **[EDA_Techniques.R](EDA_Techniques.R)**: Exploratory Data Analysis of CSV files.
8.  **[LinearRegression.R](LinearRegression.R)**: Linear and Multi-Linear Regression.
