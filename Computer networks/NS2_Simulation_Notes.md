# NS2 Network Simulator Lab

Simulation of packet drops, congestion, and performance analysis.

### 📜 Simulation Scenarios

1.  **Finding Dropped Packets**: Trace analysis to identify total packets dropped in a topology.
2.  **TCP/UDP Drop Comparison**: Analyzing the reliability difference in transport protocols.
3.  **Congestion Visualization**: Plotting throughput and data rates using `xgraph`.
4.  **Performance Determination**: Measuring delay and jitter with respect to packet transmission.
