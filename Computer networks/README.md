# Computer Networks Lab

C programs implementing data link layer protocols, routing algorithms, and network security.

### 📜 Master Program List

1.  **[FramingMethods.c](FramingMethods.c)**: Character stuffing and bit stuffing techniques.
2.  **[CRC_Computation.c](CRC_Computation.c)**: Error detection using CRC-12, CRC-16, and CRC-CCITT.
3.  **[SlidingWindow.c](SlidingWindow.c)**: Flow control using Sliding Window and Go-Back-N.
4.  **[DijkstraPaths.c](DijkstraPaths.c)**: Shortest path computation in a network.
5.  **[BroadcastTree.c](BroadcastTree.c)**: Obtaining a broadcast tree for a subnet.
6.  **[DistanceVector.c](DistanceVector.c)**: Distance Vector Routing algorithm.
7.  **[DataEncryption.c](DataEncryption.c)**: Simple data encryption and decryption.
8.  **[LeakyBucket.c](LeakyBucket.c)**: Congestion control using the Leaky Bucket algorithm.
9.  **[FrameSorting.c](FrameSorting.c)**: Techniques for sorting frames in buffers.
10. **[Wireshark_Notes.md](Wireshark_Notes.md)**: Guide for capturing traffic with Wireshark.
11. **[Nmap_Notes.md](Nmap_Notes.md)**: Commands for Nmap scans and OS detection.
12. **[NS2_Simulation_Notes.md](NS2_Simulation_Notes.md)**: Setup for the NS2 simulator.
