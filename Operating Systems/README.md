# Operating Systems Lab

A collection of C programs simulating core concepts of modern operating systems.

### 📜 Master Program List

1.  **`CPUScheduling.c`**: Implementation of FCFS, SJF, Round Robin, and Priority scheduling algorithms.
2.  **`IOSystemCalls.c`**: Demonstrating UNIX I/O system calls including `open`, `read`, `write`, `close`, `stat`, etc.
3.  **`BankersAlgorithm.c`**: Simulation of the Banker's Algorithm for deadlock avoidance.
4.  **`ProducerConsumer.c`**: Solving the Producer-Consumer synchronization problem using semaphores and system calls.
5.  **`IPCDemo.c`**: Demonstrating Inter-Process Communication (IPC) via Pipes, FIFOs, Message Queues, and Shared Memory.
6.  **`MemoryManagement.c`**: Simulation of Paging and Segmentation memory techniques.
7.  **`PageReplacement.c`**: Implementation of FCFS, LRU, and Optimal page replacement policies.
