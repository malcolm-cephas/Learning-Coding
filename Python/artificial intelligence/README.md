# Artificial Intelligence (Python Lab)

A collection of AI algorithms, search strategies, and classic game logic implemented in Python.

### 📜 Master Program List

1.  **[BFS_Graph.py](BFS_Graph.py)**: Implementation of Breadth-First Search for graph traversal.
2.  **[DFS_Graph.py](DFS_Graph.py)**: Implementation of Depth-First Search for deeper state exploration.
3.  **[TicTacToe.py](TicTacToe.py)**: Human-vs-AI or User-based Tic-Tac-Toe game logic.
4.  **[Puzzle8_Solver.py](Puzzle8_Solver.py)**: Solving the classic 8-Puzzle sliding tile problem.
5.  **[WaterJugAI.py](WaterJugAI.py)**: Solving the Water-Jug state-space problem iteratively.
6.  **[TSP_Solver.py](TSP_Solver.py)**: Heuristic or recursive solution for the Travelling Salesman Problem.
7.  **[TowerOfHanoi.py](TowerOfHanoi.py)**: Recursive solution for the disks and pegs logic.
8.  **[MonkeyBananaAI.py](MonkeyBananaAI.py)**: Rule-based simulation of the Monkey and Banana problem.
9.  **[AlphaBetaPruning.py](AlphaBetaPruning.py)**: Optimized Minimax algorithm for decision-making in games.
10. **[Queens8_Problem.py](Queens8_Problem.py)**: N-Queens backtracking solver (configured for N=8).
