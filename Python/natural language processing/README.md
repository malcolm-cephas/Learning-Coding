# Natural Language Processing (Python Lab)

A collection of NLP scripts demonstrating text processing, linguistics, and speech-to-text utilities.

### 📜 Master Program List

1.  **[TokenizeStopWords.py](TokenizeStopWords.py)**: Manual and NLTK-based Tokenization and Stop word removal.
2.  **[PorterStemmer.py](PorterStemmer.py)**: Implementation of the Porter Stemmer algorithm for linguistic root extraction.
3.  **[WordAnalysisGen.py](WordAnalysisGen.py)**: Analyzing and generating words based on morphological rules.
4.  **[WSD_Algorithm.py](WSD_Algorithm.py)**: Word Sense Disambiguation for ambiguous terms (e.g., Bank, Bat, Crane).
5.  **[NLTK_Stemming.py](NLTK_Stemming.py)**: Using the NLTK toolkit for efficient stemming operations.
6.  **[POSTagging.py](POSTagging.py)**: Part-of-Speech tagging for a list of words and single word lookup.
7.  **[NGramsAnalysis.py](NGramsAnalysis.py)**: Morphological analysis, N-Grams generation, and Smoothing logic.
8.  **[AudioTextConverter.py](AudioTextConverter.py)**: Converting audio to text and text back into audio files.
