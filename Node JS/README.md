# Web and Node.js Development Lab

A collection of web applications, Node.js servers, React components, and other full-stack technologies.

### 📜 Master Program List

1.  **[ShoppingApp/](ShoppingApp/)**: A responsive shopping cart application with registration, login, catalog, and cart pages using CSS Flexbox and Grid.
2.  **[ClientValidation/](ClientValidation/)**: Implementation of robust JavaScript client-side validation for basic web forms.
3.  **[WeatherES6/](WeatherES6/)**: Demonstration of ES6 features (Promises, Async/Await) and a weather application using the OpenWeatherMap API and Chart.js.
4.  **[JavaDatabaseCRUD/](JavaCRUD/)**: A Java standalone application demonstrating CRUD operations with a database (Oracle/MySQL).
5.  **[BookstoreXML/](BookstoreXML/)**: XML schema define for a bookstore, validated with both DTD and XSD.
6.  **[CustomServer.js](CustomServer.js)**: A Node.js custom server using the `http` module and exploring `os`, `path`, and `event` modules.
7.  **[StudentREST/](StudentREST/)**: An Express.js web application interacting with a REST API to perform CRUD operations on student data (Postman-ready).
8.  **[ReactWeather/](ReactWeather/)**: A React service that fetches current and historical weather data from OpenWeatherMap using Chart.js.
9.  **[ReactTodo/](ReactTodo/)**: A complete React TODO application with state management and component structure.
