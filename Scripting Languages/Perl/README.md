# Perl Scripting Lab

A collection of Perl scripts for text processing, list manipulation, and data validation using regex.

### 📜 Master Program List

1.  **[largest_num.pl](largest_num.pl)**: Logic to find the largest of three input numbers.
2.  **[tables_print.pl](tables_print.pl)**: Prints multiplication tables for numbers 1 to 10.
3.  **[list_functions.pl](list_functions.pl)**: Demonstrating `shift`, `unshift`, and `push` list functions.
4.  **[replace_str.pl](replace_str.pl)**: Using `substr` to substitute strings based on indexing.
5.  **[validate_ipaddr.pl](validate_ipaddr.pl)**: Regex-based validation of IP and email addresses.
6.  **[reverse.pl](reverse.pl)**: Reverses the order of lines in a text file.
