# Ruby Scripting Lab

A collection of basic Ruby scripts for mathematical, logical, and string operations.

### 📜 Master Program List

1.  **[str_n_copies.rb](str_n_copies.rb)**: Returns *n* copies of a string.
2.  **[perimeter_area.rb](perimeter_area.rb)**: Calculates circle perimeter (circumference) and area.
3.  **[reverse_names.rb](reverse_names.rb)**: Greets the user with their last name first.
4.  **[file_extension.rb](file_extension.rb)**: Extracts the file extension from a filename.
5.  **[greatest_num.rb](greatest_num.rb)**: Finds the greatest of three numbers.
6.  **[odd_nums.rb](odd_nums.rb)**: Prints odd numbers from 10 down to 1.
7.  **[check_num.rb](check_num.rb)**: Function that returns `true` if one of two numbers is 20 or if their sum is 20.
8.  **[check_temperatures.rb](check_temperatures.rb)**: Checks if one temp is < 0 and the other > 100.
9.  **[array_elements.rb](array_elements.rb)**: Iterates through and prints array elements.
10. **[total_marks.rb](total_marks.rb)**: Calculates total marks from a hash of subjects.
