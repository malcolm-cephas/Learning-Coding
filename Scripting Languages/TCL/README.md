# TCL Scripting Lab

A collection of TCL scripts for algorithmic and file-handling logic.

### 📜 Master Program List

1.  **[factorial.tcl](factorial.tcl)**: Iterative factorial calculation in TCL.
2.  **[factorial_recur.tcl](factorial_recur.tcl)**: Recursive factorial implementation.
3.  **[multiplies_nums.tcl](multiplies_nums.tcl)**: Procedure to multiply numbers from 1 to 10.
4.  **[sort_list.tcl](sort_list.tcl)**: A custom procedure to sort elements in a list.
5.  **[list_ops.tcl](list_ops.tcl)**: Demonstrating TCL list operations (create, append, traverse, concat).
6.  **[compare_files.tcl](compare_files.tcl)**: Compares modification times of two files.
7.  **[copy_translate.tcl](copy_translate.tcl)**: Copies contents between files with line translation.
