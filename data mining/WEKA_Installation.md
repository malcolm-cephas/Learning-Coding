# WEKA Installation and Features Guide

### ⬇️ Installation (Exp 1)
- Go to the [Waikato University WEKA page](https://www.cs.waikato.ac.nz/ml/weka/).
- Download the version corresponding to your OS (Mac, Windows, Linux).
- Ensure Java 1.8+ is installed.

### 📊 Understand WEKA Features (Exp 2)
The WEKA Explorer environment provides:
1.  **Preprocess**: Loading and filtering data.
2.  **Classify**: Supervised learning with algorithms like J48 (Decision Trees), Naive Bayes, and Random Forest.
3.  **Cluster**: Unsupervised learning for finding patterns.
4.  **Associate**: Discovering association rules (e.g., Apriori).
5.  **Visualize**: 2D plotting of attributes.

### 📂 Loading Data Sources (Exp 3)
- **Open File**: Choose a `.arff` or `.csv` from your local disk.
- **Open URL**: Enter a remote dataset link.
- **Open DB**: Use JDBC to query SQL databases directly.
