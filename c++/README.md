# C++ Object Oriented Programming Lab (2018-2019)

Master list of C++ programs covering basic logic, classes, inheritance, templates, and exception handling.

### 📜 Master Program List

#### Week 2: Basic Logic
*   **[Week2_a_SumDigits.cpp](Week2_a_SumDigits.cpp)**: Sum of individual digits of a positive integer.
*   **[Week2_b_Fibonacci.cpp](Week2_b_Fibonacci.cpp)**: Generate the first n terms of a sequence (Fibonacci).

#### Week 3: Prime Numbers & Search
*   **[Week3_a_Primes.cpp](Week3_a_Primes.cpp)**: Generate all prime numbers between 1 and n.
*   **[Week3_b_MinMax.cpp](Week3_b_MinMax.cpp)**: Find both the largest and smallest number in a list.

#### Week 4: Sorting & Memory
*   **[Week4_a_Sort.cpp](Week4_a_Sort.cpp)**: Sort a list of numbers in ascending order.
*   **[Week4_b_DynamicMemory.cpp](Week4_b_DynamicMemory.cpp)**: Illustrate `new` and `delete` keywords for dynamic allocation.

#### Week 5: Classes & Objects
*   **[Week5_a_ClassDemo.cpp](Week5_a_ClassDemo.cpp)**: Class Declaration, Definition, and Member Access.
*   **[Week5_b_Constructors.cpp](Week5_b_Constructors.cpp)**: Default, Parameterized, and Copy constructors.
*   **[Week5_c_StudentClass.cpp](Week5_c_StudentClass.cpp)**: Student class with marks calculation and display.

#### Week 6: Overloading & Friendship
*   **[Week6_a_Overloading.cpp](Week6_a_Overloading.cpp)**: Operator Overloading and Function Overloading.
*   **[Week6_b_FriendDemo.cpp](Week6_b_FriendDemo.cpp)**: Friend Function and Friend Class demonstration.

#### Week 7: Pointers & Initialization
*   **[Week7_a_PointerToObject.cpp](Week7_a_PointerToObject.cpp)**: Accessing members using Pointer to Object.
*   **[Week7_b_FibonacciConstructor.cpp](Week7_b_FibonacciConstructor.cpp)**: Fibonacci series initialized via Constructor.

#### Week 9: Matrix ADT
*   **[Week9_MatrixADT.cpp](Week9_MatrixADT.cpp)**: Matrix operations (Add, Sub, Mult, Read, Print) using a class.

#### Week 10: Inheritance
*   **[Week10_Inheritance.cpp](Week10_Inheritance.cpp)**: Single, Multiple, Multilevel, and Hierarchical inheritance.

#### Week 11: Polymorphism & Lifecycle
*   **[Week11_a_CtorDtorOrder.cpp](Week11_a_CtorDtorOrder.cpp)**: Execution order of constructors/destructors in multiple inheritance.
*   **[Week11_b_BasePointer.cpp](Week11_b_BasePointer.cpp)**: Invoking derived class members through Base Class Pointer.

#### Week 12: Templates
*   **[Week12_a_TemplateSort.cpp](Week12_a_TemplateSort.cpp)**: Template-based sorting for general types.
*   **[Week12_b_TemplateMinMax.cpp](Week12_b_TemplateMinMax.cpp)**: Function templates for search and sort operations.

#### Week 13: Exception Handling
*   **[Week13_a_ExceptionDemo.cpp](Week13_a_ExceptionDemo.cpp)**: Throw and Catch basic exceptions.
*   **[Week13_b_CatchAll.cpp](Week13_b_CatchAll.cpp)**: Demonstrating the "Catch-All" exception handler.

#### Supplemental Programs
*   **[Prog_22_LargerOfTwo.cpp](Prog_22_LargerOfTwo.cpp)**: Read two numbers and display the larger value.
*   **[Prog_23_SumNatural.cpp](Prog_23_SumNatural.cpp)**: Compute sum of natural numbers from 1 to n.
*   **[Prog_24_CallByReference.cpp](Prog_24_CallByReference.cpp)**: Swap two values using call by reference.
*   **[Prog_25_FunctionOverloading.cpp](Prog_25_FunctionOverloading.cpp)**: Demonstrate overloaded `add` function (int and double).
*   **[Prog_26_ShapeInheritance.cpp](Prog_26_ShapeInheritance.cpp)**: Complex inheritance (Shape-Polygon-Rectangle-Square).
*   **[Prog_27_VehicleInheritance.cpp](Prog_27_VehicleInheritance.cpp)**: Multilevel inheritance (Vehicle-FourWheeler-Car).
*   **[Prog_28_FileIOBasic.cpp](Prog_28_FileIOBasic.cpp)**: Create, check, write to, and read from a text file.
*   **[Prog_29_BinaryFileTime.cpp](Prog_29_BinaryFileTime.cpp)**: Binary file I/O operations for time objects.
*   **[Prog_30_DivideByZeroException.cpp](Prog_30_DivideByZeroException.cpp)**: Standard division by zero exception handling.
*   **[Prog_31_ArrayBoundsException.cpp](Prog_31_ArrayBoundsException.cpp)**: Catching array index out of bounds exceptions.
