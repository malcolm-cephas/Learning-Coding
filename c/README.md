# C Programming Lab

A collection of foundational C programs for mathematical computing, data sorting, and pointer-based statistics.

### 📜 Master Program List

1.  **[MechanicalEnergy.c](MechanicalEnergy.c)**: Compute energy using $E = mgh + 0.5mv^2$.
2.  **[KmConverter.c](KmConverter.c)**: Conversion of Kilometers into Meters and Centimeters.
3.  **[CharClassification.c](CharClassification.c)**: Detect Lowercase, Uppercase, or Special Characters.
4.  **[BalancingEquation.c](BalancingEquation.c)**: Simple chemical equation balancing logic.
5.  **[MatrixMultValidation.c](MatrixMultValidation.c)**: Matrix multiplication with rule validation ($C_{ij} = \sum A_{ik} B_{kj}$).
6.  **[TaylorTrigApprox.c](TaylorTrigApprox.c)**: Sin(x)/Cos(x) approximation via Taylor series.
7.  **[BubbleSort.c](BubbleSort.c)**: Sorting N numbers using the Bubble sort algorithm.
8.  **[StringHandmadeOps.c](StringHandmadeOps.c)**: Custom string functions (Compare, Concat, Length).
9.  **[StudentStructAverage.c](StudentStructAverage.c)**: Struct-based student records, marks calculation, and average detection.
10. **[PointerStats.c](PointerStats.c)**: Using pointers to compute Sum, Mean, and Standard Deviation.
